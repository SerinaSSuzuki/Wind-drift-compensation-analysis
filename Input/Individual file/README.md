There are two types of input files: a GPS file and an individual file. This is the folder for individual file.
Individual file includes "id"	"foot"	"age"	"sex"	"logger"	"type"	"year"
There are two types of ID: "id" and "foot". The "id" format follows a pattern such as K13B27, where K13 refers to the collection site and year — for example, K13 indicates data collected at Kabushima in 2013 — and B27 refers to the individual ID, which may change each year.
In contrast, "foot" refers to the metal leg band number attached to each individual through mark-recapture. This number remains constant throughout the individual's life.
"logger" and "type" refers to which logger used to collect the GPS data, and how it is attached: Tape or Harness.
